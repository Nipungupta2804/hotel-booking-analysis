# Hotel Booking Performance Analysis

## 📌 Problem
Analyze hotel booking data to understand booking trends, revenue distribution, and hotel performance.

## 🛠️ Tools Used
- Power BI

## 📊 Dataset
- 50,000+ booking records
- Includes bookings, hotels, users, and transaction data

## ⚙️ Work Done
- Performed data cleaning and transformation in Power BI (Power Query)
- Built data model with relationships between tables
- Created DAX measures:
  - Total Revenue
  - Total Bookings
  - Occupancy Rate
  - Revenue per Booking
- Developed interactive dashboards with filters (city, date, room type)

## 🔍 Key Insights
- Revenue is concentrated among top-performing hotels
- Certain cities contribute more to overall revenue
- Some high-capacity hotels have low occupancy → underutilization
- Revenue varies across hotels despite similar booking volumes

## 📈 Dashboard Features
- Booking trend analysis over time
- Hotel-wise performance comparison
- City-wise revenue distribution
- Occupancy and utilization insights
